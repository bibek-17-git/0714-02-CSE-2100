# Build Instructions

To set up the build environment for this project, follow the instructions below:

## Prerequisites
- Ensure you have [Git](https://git-scm.com) installed.
- Install [Node.js](https://nodejs.org/) (version x.x.x) - check the compatibility in the repository.
- Install [npm](https://www.npmjs.com/) (comes along with Node.js).

## Installing Dependencies

Run the following command in the project root directory:
```bash
npm install
```

## Running the Build

To build the project, use the command:
```bash
npm run build
```

## Running Tests

To ensure everything is working correctly, run:
```bash
npm test
```
